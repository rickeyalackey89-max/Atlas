# scripts/dev

Non-runtime scripts and experiments. Nothing here should be required for `run_today.py`.

## Folders

- `boards/` — board snapshots, updates, compares, audits
- `gamelogs/` — update/refresh gamelogs utilities
- `validation/` — sanity checks, playability validation
- `analysis/` — analysis and exploration scripts
- `export/` — exports to dashboards / external systems
- `adhoc/` — one-off utilities that don't fit elsewhere
- `diagnostics/` — debug and diagnostic helpers
